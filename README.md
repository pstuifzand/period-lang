period-lang
===========

Demo of date/period specific language

Description
-----------

Perl Demo library for working with a date periods in a small language.

Example
-------

Expression:

    2013-03-22 between [2012-01-01,2012-12-31]

Multiple statements:

    p = [2012-01-01,];
    q = [2013-12-01,2013-12-31];
    p overlaps q;
